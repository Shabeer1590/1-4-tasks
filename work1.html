What is a browser?

A browser is a software application that allows you to access and view websites on the internet

What is internet?

The Internet is a global network of interconnected computer networks that uses the Internet Protocol Suite (TCP/IP) to communicate

What is html?

HTML, short for HyperText Markup Language, is the foundation of the World Wide Web, defining the structure and content of web pages.

What is xml?

XML stands for extensible markup language. A markup language is a set of codes, or tags, that describes the text in a digital document.

What is http?

HTTP, or Hypertext Transfer Protocol, is the foundational protocol of the World Wide Web, defining how web servers and clients (like your browser) communicate.

What is https?

HTTPS stands for Hypertext Transfer Protocol Secure.
